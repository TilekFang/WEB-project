DSTIKE-Deauther-Watch

Discription:

1. Deauther Watch: It is an ESP8266 development board, but you can wear it like a smartwatch, more convenient. Usuage: Using it only for testing and educational purposes against your own devices!
2. Wifi Testing Tool: This esp8266 dev board is installed with the latest ESP8266 Deauther software. With this software, you can perform different secure tests for your WiFi networks. NOTE: ESP8266 supports 2.4GHz only!
3. Long Working Time: Built-in an 500mAh lithium battery, working time up to 7 hours. Comes with OLED display and a 3 way momentary slide switch.
4. Extension Board: You can use it to develop your own software. It is indeed a ESP8266 development board and easy to operate.
